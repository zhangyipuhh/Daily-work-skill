# 其他过程文档 大纲模板

> 适用：周报、会议纪要、检查单、风险登记册等其他过程文档
> 评审：通常不需要正式评审

## 周报大纲

1. **本周工作完成情况**
2. **下周工作计划**
3. **存在问题与风险**
4. **需要协调事项**
5. **关键数据**（进度/成本/缺陷）

## 会议纪要大纲

1. **会议基本信息**（时间/地点/主持/记录/出席/缺席）
2. **议题与讨论**
3. **决议事项**
4. **行动项**（负责人/截止日期）
5. **遗留问题**

## 检查单大纲

1. **检查范围**
2. **检查依据**
3. **检查项**（编号/项/标准/结果/备注）
4. **检查结论**
5. **整改建议**

## 风险登记册大纲

1. **风险概述**
2. **风险项**（编号/描述/概率/影响/应对/责任人/状态）
3. **风险趋势**
4. **已关闭风险**
