## 模糊匹配关键词

> 子标签名称可能因项目而异，建议**模糊匹配**以下关键词：

| 标准子标签 | 备选名称 |
|---|---|
| 总体计划概述 | 项目概述 / Overview / 项目概况 |
| 里程碑 | Milestone / 关键节点 / 节点计划 |
| 评审计划 | 评审安排 / Review Plan / 评审 |
| 风险登记册 | 风险 / Risk Register / 风险清单 |
| 汇总损益分析 | 损益 / Cost Summary / 成本汇总 |
| WBS | 工作分解 / 任务分解 / 任务清单 |
| RACI | 责任矩阵 / 责任分配 / 责任人 |

## 提取建议

- 用 openpyxl 读取整个 sheet 内容，按 `\t` 分隔列
- 对每行扫描"列头"，定位关键列
- 找不到字段 → 提示用户上传策划表或确认策划表版本
- 找到字段 → 输出"行号+列名"作为证据，便于追溯
