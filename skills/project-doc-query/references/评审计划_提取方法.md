# 评审计划提取方法

> 用户问"什么时候交/什么时候评审"时的标准提取流程。

## Step 1：定位策划表

```python
from pathlib import Path
import glob

project_root = Path(r"D:\项目文档\202410-C0008-...")
candidate = sorted(
    glob.glob(str(project_root / "01_项目策划" / "*策划表*V*.xlsm")),
    key=lambda p: [int(s) for s in Path(p).stem.split("V")[-1].split(".") if s.isdigit()],
    reverse=True,
)
plan_file = candidate[0]  # 取版本最大
```

## Step 2：读"评审计划"sheet（用 CLI 脚本，禁止 Python import）

```bash
# 方式 A：用 find_planning_sheet.py 一条命令完成"定位 + 读评审计划"
# 先定位策划表
python scripts/find_planning_sheet.py --project-root "D:\项目文档\202410-C0008-..." --output path
# → 输出: D:\...\策划表V1.0.xlsm

# 再读"评审计划"sheet（自动模糊匹配：评审 / Review / Plan）
python scripts/read_doc.py --file "<上一步输出>" --sheet 评审计划 --output table
```

完整参数说明见 `references/cli_脚本使用速查.md`。

## Step 3：解析行

- 列：`交付物 | 提交日期 | 评审日期 | 评审方式 | 评审人 | 评审结论`
- 按 `\t` 分隔每行
- 渲染为表格输出

## 输出格式

| 交付物 | 提交日期 | 评审日期 | 评审方式 | 评审人 | 评审结论 |
|---|---|---|---|---|---|
| 需求说明书 | 2025-12-10 | 2025-12-15 | 会议评审 | 张三、李四 | 通过 |
| ... | ... | ... | ... | ... | ... |

## 异常处理

| 异常 | 处理 |
|---|---|
| 找不到策划表 | 提示用户提供策划表位置 |
| xlsm 读取失败（文件被锁） | 提示关闭 Excel 后重试 |
| 找不到"评审计划"子标签 | 用模糊匹配"评审"关键词 |
| 评审计划为空 | 询问用户是否有评审计划 |
| 字段错位 | 提示用户上传预览截图或手动确认 |
