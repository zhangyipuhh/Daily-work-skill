# 评审计划提取方法

> 用户问"什么时候交/什么时候评审"时的标准提取流程。

## Step 1：定位策划表

```python
from pathlib import Path
import glob

project_root = Path(r"D:\项目文档\202410-C0008-...")
candidate = sorted(
    glob.glob(str(project_root / "01_项目策划" / "*策划表*V*.xlsm")),
    key=lambda p: [int(s) for s in Path(p).stem.split("V")[-1].split(".") if s.isdigit()],
    reverse=True,
)
plan_file = candidate[0]  # 取版本最大
```

## Step 2：读"评审计划"sheet

```python
# skill 自带的 DocumentLoader（独立自包含，无外部包依赖）
# 位置：本 skill 的 scripts/ 下
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).resolve().parents[1] / "scripts"))  # 指向 skill/scripts
from DocumentLoader import DocumentLoader
from loader.ExcelLoader import ExcelLoader

loader = DocumentLoader(plan_file, custom_kwargs={".xlsm": {"data_only": True}})
docs = loader.load_with_config(".xlsm")
# docs[i].metadata["sheet_name"] == "评审计划" 时取该 doc
```

## Step 3：解析行

- 列：`交付物 | 提交日期 | 评审日期 | 评审方式 | 评审人 | 评审结论`
- 按 `\t` 分隔每行
- 渲染为表格输出

## 输出格式

| 交付物 | 提交日期 | 评审日期 | 评审方式 | 评审人 | 评审结论 |
|---|---|---|---|---|---|
| 需求说明书 | 2025-12-10 | 2025-12-15 | 会议评审 | 张三、李四 | 通过 |
| ... | ... | ... | ... | ... | ... |

## 异常处理

| 异常 | 处理 |
|---|---|
| 找不到策划表 | 提示用户提供策划表位置 |
| xlsm 读取失败（文件被锁） | 提示关闭 Excel 后重试 |
| 找不到"评审计划"子标签 | 用模糊匹配"评审"关键词 |
| 评审计划为空 | 询问用户是否有评审计划 |
| 字段错位 | 提示用户上传预览截图或手动确认 |
