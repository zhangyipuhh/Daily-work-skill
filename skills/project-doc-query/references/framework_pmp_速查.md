## 5 大过程组

| 过程组 | 中文 | 核心问题 |
|---|---|---|
| Initiating | 启动 | 项目授权、识别干系人 |
| Planning | 规划 | 制定行动路线 |
| Executing | 执行 | 完成项目管理计划 |
| Monitoring & Controlling | 监控 | 跟踪、审查、调整 |
| Closing | 收尾 | 正式结束项目或阶段 |

## 套件使用建议

- 用户问"项目管理体系长什么样" → 用 PMP 框架层
- 用户问"我应该做什么" → 用 PMP 找对应知识领域
- 用户问"具体怎么做" → 切到 PRINCE2 落地层
